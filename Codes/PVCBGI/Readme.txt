1. "PVCB.dll";
The generated 32-bit DLL file which emulates a controlled PV system including the F-P and V-Q droop power outer loop, the virtual synchronous generator controller and the P-I voltage controller (The coefficiencies of the P-I voltage controller has not been tunned).

2. "DelphiCodes";
The folder includes the Delphi codes where the "PVModel.pas" is the file stores the method to calculate the terminal current and state variables.

3. "IEEE13TEST.dss";
The OpenDSS file has a IEEE 13 nodes test case where the established PV system model is employed in the 634 node.

4. "IEEE13Node_BusXY.csv";
The file storing the coordinates of the test case.
